# FEATURES.md — ميزات البوت

## 🪅 النظام

| الميزة | الوصف |
|--------|-------|
| **قائمة ديناميكية** | تُحسب الأوامر وعددها تلقائياً من الـ Registry عند كل طلب |
| **توثيق ذاتي** | يُنشئ `docs/COMMANDS.md` تلقائياً عند بدء التشغيل |
| **البادئة القابلة للتغيير** | يمكن تغييرها وقت التشغيل، تُحفظ في MongoDB |
| **معلومات النظام** | عرض RAM، CPU، وقت التشغيل، إصدار Node.js |
| **Multi-account** | يدعم حسابين Facebook في آن واحد |
| **Auto-reconnect** | يعيد الاتصال تلقائياً عند انقطاع MQTT |
| **Health check** | نقطة `/health` لـ Railway healthcheck |

## 🌨 الخاصة

| الميزة | الوصف |
|--------|-------|
| **إرسال تلقائي** | إرسال رسالة متكررة بفاصل زمني محدد لكل قروب |
| **تحميل الأغاني** | بحث في يوتيوب وإرسال الصوت كـ MP3 مع cooldown |
| **حفظ الإعدادات** | يحفظ إعدادات كل قروب في MongoDB مع fallback لملف محلي |

## 🏴 الإدارة

| الميزة | الوصف |
|--------|-------|
| **طرد الأعضاء** | طرد بالرد على رسالة أو بالـ ID مع تأكيد |
| **إضافة أعضاء** | إضافة عضو بالـ ID مع عرض اسمه |
| **إدارة الأدمن** | عرض الأدمن وإضافتهم وحذفهم |
| **إغلاق القروب** | تجاهل أوامر غير الأدمن مع حفظ الحالة |
| **إدارة القروبات** | عرض كل القروبات مع إمكانية الرسالة/المغادرة/التسمية/الكتم |
| **الطلبات المعلقة** | عرض PENDING + SPAM + OTHER + ARCHIVED وقبولها/رفضها |
| **حماية الاسم** | إعادة اسم القروب تلقائياً عند أي محاولة تغييره |
| **حماية الكنيات** | إعادة كنيات الأعضاء تلقائياً |
| **اسم البوت المحمي** | حماية كنية البوت من التغيير |

## 🛡 الحماية والأمان

| الميزة | الوصف |
|--------|-------|
| **Anti-Spam** | حدّ أقصى 5 رسائل في 10 ثواني |
| **Cooldown** | فترة انتظار 3 ثوانٍ بين كل أمر (قابل للتخصيص لكل أمر) |
| **Permissions** | نظام صلاحيات: user → moderator → admin → owner |
| **Ban System** | حظر مؤقت أو دائم مع سبب، محفوظ في MongoDB |
| **Lockdown** | إغلاق قروب بالكامل أمام غير الأدمن |
| **Credential Encryption** | تشفير appState عند التخزين |
| **Session Management** | إدارة جلسات آمنة مع TTL |

## ⚙️ البنية التقنية

| الميزة | الوصف |
|--------|-------|
| **Plugin System** | إضافة أوامر جديدة دون تعديل الكود الأساسي |
| **Hot Reload** | إعادة تحميل الأوامر عند تغيير الملفات (dev mode) |
| **MongoDB + Fallback** | يحفظ في MongoDB، يرجع لملف محلي عند غيابه |
| **Redis Cache** | دعم Redis اختياري للـ caching |
| **Task Scheduler** | مهام مجدولة recurring + delayed |
| **Structured Logging** | لوغ منظّم مع مستويات (info/warn/error) |
| **TypeScript Strict** | كود مكتوب بـ TypeScript strict mode |
